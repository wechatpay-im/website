| 返回字典 | 类型 {.type} | 描述 {.desc}

| return_code {data-required} | string | 返回状态码<br/>`SUCCESS` \| `FAIL` 枚举值之一
| return_msg | string | 返回信息

| err_code_des | string | 错误代码描述

| plan_id | string | 签约模板id
